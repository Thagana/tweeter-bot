# twitter-bot

python twitter bot quotes using tweepy and wikiqoutes

## Getting Started

```bash
    $ git clone https://github.com/Thagana/twitter-bot.git
```

```bash
    $ cd twitter-bot
```

```bash
    $ virtualenv env
```

```bash
    $ pip install -r requirements.txt
```

```bash
    $ python tweet-quotes.py
```
